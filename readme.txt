== Installation ==

1. You have a couple of options:
	* Go to Plugins &rarr; Add New and search for "Alpha Blocks". Once found, click "Install".
	* Download the Alpha Blocks from wordpress.org and make sure the folder is zipped. Then upload via Plugins &rarr; Add New &rarr; Upload.
    * Open the block inserter within the Block Editor (Gutenberg) and search for "icon". The plugin should appear and allow you to install it directly. Skip steps 2 and 3.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Search for the block within the Block Editor (Gutenberg) and begin using it.


== Frequently Asked Questions ==

An answer to that question.

= What is Alpha Blocks, and how does it enhance Gutenberg editing? =

Alpha Blocks is a WordPress plugin designed to augment the Gutenberg editor by providing a diverse collection of customizable content blocks. These blocks offer dynamic layouts and interactive elements, empowering users to elevate their content creation experience seamlessly within the Gutenberg environment.

= Can I use Alpha Blocks if I don't have coding skills? =

Absolutely! Alpha Blocks is developed with a user-friendly interface, making it accessible to both beginners and seasoned developers. You can effortlessly customize each block without the need for coding skills, allowing you to tailor your website's look and feel according to your preferences.

= Is Alpha Blocks optimized for performance? =

Yes, indeed. Alpha Blocks is designed to be lightweight, ensuring optimal website performance. This means fast loading times and a responsive design across various devices. You can enhance your website with dynamic content without compromising on speed or efficiency.
